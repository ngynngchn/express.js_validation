# SuperCode Logistics

Wir bauen den ultimativen Losgistiksimulator auf.
Version 1.0:

- [ ] Wir haben eine Route um neue LKW's anzulegen
- [ ] Wir haben eine Route um alle LKW's auszugeben

### Datenmodel

**LKW**

- Fahrgestellnummer(number)
- Hersteller (string)
- Kilometerstand (number)
- Kennzeichen
- Last (40t, 7,5t)
- Farbe
